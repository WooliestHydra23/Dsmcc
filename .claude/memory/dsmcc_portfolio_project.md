## Key Decisions Made

### Architecture Choice: Jekyll, Not React
**Decision**: Keep Jekyll + GitHub Pages instead of converting to React  
**Why**: Fast shipping, free hosting, maintainable, still looks modern with CSS/JS animations  
**Trade-off**: Some advanced features harder (but not necessary)  

### Animation Inspiration: Soumyajit Portfolio
**Decision**: Adapted visual design patterns from soumyajit4419/Portfolio  
**Implements**: Particles, typewriter, glassmorphism, smooth animations  
**Why**: Proven effective design, industry-standard modern aesthetic  

### Content Strategy: Bridge, Not Pure Technical
**Decision**: Equal weight to business projects + technical code samples  
**Why**: Reflects Dron's actual unique value—not just strategy, not just code  

### Deployment: GitHub Pages Only (No Custom Domain Yet)
**Decision**: Use wooliesthydra23.github.io/Dsmcc for now  
**Why**: Fast to set up, free, sufficient for resume/hiring  
**Future**: Can add custom domain in Phase 4 if needed
